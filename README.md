# CyberGuard - Cybersecurity Awareness Chatbot

## Description
A WPF C# chatbot that educates users about cybersecurity topics.

## Features
- GUI Design with dark theme and ASCII art
- Keyword Recognition (passwords, phishing, scams, privacy, malware)
- Random Responses using Dictionary and List
- Conversation Flow with follow-up questions
- Memory and Recall (remembers user name and interests)
- Sentiment Detection (worried, happy, curious, frustrated)
- Error Handling for unknown inputs
- Code Optimisation using OOP, Dictionary, List

## How to Run
1. Open `CyberAwarenessBot.sln` in Visual Studio 2022
2. Press F5 to run

## Tech Stack
- C# / WPF (.NET 8.0)
- Visual Studio 2022
